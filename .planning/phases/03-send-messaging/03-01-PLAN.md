**3. Create `src/services/rate-limiter.ts` (per D-05, D-06, D-07):**

Export a `TokenBucket` class with:
- Constructor: `constructor(capacity: number, refillPerHour: number)` -- sets `this.tokens = capacity`, `this.lastRefill = Date.now()`, `this.sendCount = 0`.
- `consume(): boolean` -- calls `this.refill()`, returns `false` if `this.tokens < 1`, else decrements tokens and returns `true`.
- Private `refill(): void` -- calculates elapsed hours since lastRefill, adds `elapsed * refillPerHour` tokens, caps at `capacity`, updates `lastRefill`.
- `getJitterMs(): number` -- increments `this.sendCount`. Every 3-5 sends (randomize interval: `3 + Math.floor(Math.random() * 3)`), return long pause `30_000 + Math.random() * 60_000` (30-90s). Otherwise return `2_000 + Math.random() * 6_000` (2-8s). Per D-07.
- Getter `remainingTokens: number` -- calls `refill()`, returns `Math.floor(this.tokens)`.

Also export a singleton factory function:
```typescript
import { env } from '../config/env.js';

let instance: TokenBucket | null = null;

export function getRateLimiter(): TokenBucket {
  if (!instance) {
    instance = new TokenBucket(env.RATE_LIMIT_CAPACITY, env.RATE_LIMIT_REFILL_PER_HOUR);
  }
  return instance;
}
```

**4. Create `src/services/bluebubbles.ts` (per D-09, D-10, D-11):**

Export a `BlueBubblesClient` class with:
- Constructor: takes `baseUrl: string` and `password: string` parameters.
- Private `buildUrl(path: string): string` -- creates URL from `new URL(path, this.baseUrl)`, sets `password` search param, returns string.
- `async request<T>(path: string, options?: RequestInit): Promise<T>` -- calls `fetch(this.buildUrl(path), { ...options, signal: AbortSignal.timeout(10_000) })`. On fetch error (ECONNREFUSED, timeout), throw `new AppError('BlueBubbles server is unreachable', ERROR_CODES.BB_OFFLINE, true, 503)`. IMPORTANT: Do NOT log the raw error (it contains the password in the URL -- Pitfall 1). On success, parse JSON body. If `body.status !== 200`, throw `new AppError(body.error?.message ?? 'BlueBubbles request failed', ERROR_CODES.SEND_FAILED, false, 502)`. Return `body.data as T`.
- `async sendMessage(phone: string, message: string): Promise<{ guid: string; text: string }>` -- Generates `tempGuid` via `crypto.randomUUID()`. Calls `this.request('/api/v1/message/text', { method: 'POST', headers: { 'Content-Type': 'application/json' }, body: JSON.stringify({ chatGuid: \`any;-;+${phone}\`, tempGuid: \`temp-${tempGuid}\`, message }) })`. Returns the response typed as `{ guid: string; text: string }`.

Also export a singleton factory:
```typescript
import { env } from '../config/env.js';

let instance: BlueBubblesClient | null = null;

export function getBBClient(): BlueBubblesClient {
  if (!instance) {
    instance = new BlueBubblesClient(env.BLUEBUBBLES_URL, env.BLUEBUBBLES_PASSWORD);
  }
  return instance;
}
```

Import AppError from `'../types/errors.js'` and ERROR_CODES from `'../types/error-codes.js'`. Use `import type` for type-only imports per verbatimModuleSyntax.
