<task type="auto">
  <name>Task 2: ChatCache service, getChatDetails, and group-aware conversation mapping</name>
  <files>src/services/chat-cache.ts, src/services/bluebubbles.ts</files>
  <read_first>src/services/bluebubbles.ts, src/services/dedup.ts</read_first>
  <action>
1. Create `src/services/chat-cache.ts` following the DedupBuffer pattern from `src/services/dedup.ts`:
   - `class ChatCache` with private `cache: Map<string, { data: BBChat; expiresAt: number }>`
   - Constructor takes `ttlMs: number = 300_000` (5 min default)
   - `get(guid: string): BBChat | undefined` -- returns cached entry if not expired, deletes and returns undefined if expired
   - `set(guid: string, data: BBChat): void` -- stores with `expiresAt = Date.now() + ttlMs`
   - `destroy(): void` -- clears interval, clears map
   - Private `evict()` method called on `setInterval(60_000)` with `.unref()` (same pattern as DedupBuffer)
   - Export `ChatCache` class

2. In `src/services/bluebubbles.ts`:
   - Import `isGroupChat` from `../types/bluebubbles.js`
   - Import `ChatCache` from `./chat-cache.js`
   - Add private `chatCache: ChatCache` field to `BlueBubblesClient`, initialized in constructor
   - Add `async getChatDetails(chatGuid: string): Promise<BBChat>` method:
     - Check cache first: `const cached = this.chatCache.get(chatGuid); if (cached) return cached;`
     - Fetch from BB API: `this.request<BBChat>('/api/v1/chat/' + encodeURIComponent(chatGuid) + '?with[]=participants')`
     - Cache the result and return it
   - Refactor `sendMessage` signature from `(phone: string, message: string)` to `(chatGuid: string, message: string)`:
     - Remove the `any;-;${phone}` concatenation -- caller now provides the full chatGuid
     - The body still sends `{ chatGuid, tempGuid, message }`
     - This allows both 1:1 (`iMessage;-;+1234567890`) and group (`iMessage;+;chat123`) GUIDs
   - Update `mapBBChatToConversation(chat: BBChat): Conversation`:
     - Add `isGroup: isGroupChat(chat.guid)` to returned object
     - Add `groupName: chat.displayName` (display name is the group name for groups, null for 1:1)
     - Add `participants: chat.participants?.map(p => p.address) ?? []`
   - Add `destroyCache(): void` method to BlueBubblesClient that calls `this.chatCache.destroy()`
  </action>
